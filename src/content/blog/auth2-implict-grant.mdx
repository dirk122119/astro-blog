## 註冊憑證
到[Google API Console](https://console.developers.google.com/?hl=zh-tw)註冊OAuth2.0憑證

## 在應用程式端發http請求
```
https://accounts.google.com/o/oauth2/v2/auth?
response_type=token&client_id=${yourClientId}&
redirect_uri=${yourRedirectUri}&
scope=${authrieScope}
```
